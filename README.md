# Bill Splitter Web App

**Bill Splitter** is a web-based application for managing shared expenses among friends, roommates, or groups. It allows users to:

- Create and manage multiple groups (Trip, Roommates, Office lunch)
- Add, edit, and remove friends
- Split bills evenly or unevenly among friends
- Track who owes whom and total expenses per friend
- Mark bills as Paid or Pending
- Generate expense charts to visualize spending
- Export a PDF report of all bills and totals
- Generate invite links for friends to join the same group

**Technologies:** HTML, CSS, JavaScript, Chart.js, jsPDF

**Features:** Mobile-friendly, interactive dashboard, real-time expense calculation, visual summaries.
